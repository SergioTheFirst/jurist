# LegalDesk Windows Desktop Bundle

## Обзор

Пакет для создания установщика Windows для LegalDesk — рабочего места юриста.

**Возможности:**
- Единый инсталлятор (.exe) для установки приложения
- Локальный сервер запускается автоматически при установке
- Опция автозапуска при входе в систему
- Ярлык на рабочем столе для быстрого доступа
- Отдельный ярлык для остановки сервера
- Корректная остановка сервера при удалении

## Требования для сборки

1. **Python 3.10+** — должен быть установлен в системе или в `.venv`
2. **Inno Setup 6** — для компиляции установщика
   - Скачать: https://jrsoftware.org/isdl.php
3. **PyInstaller** — устанавливается через pip

## Подготовка окружения

```powershell
# Создать виртуальное окружение (если нет)
python -m venv .venv

# Активировать и установить зависимости
.venv\Scripts\activate.ps1
pip install -r requirements.txt
pip install pyinstaller
```

## Сборка установщика

Запустите скрипт сборки из корня проекта:

```powershell
.\packaging\windows\build_installer.ps1
```

Скрипт выполнит:
1. Найдёт Python (из `.venv` или системный)
2. Найдёт Inno Setup Compiler (ISCC.exe)
3. Извлечёт версию из `backend/main.py`
4. Очистит временные директории (`build`, `dist`, `dist-installer`)
5. Запустит PyInstaller для создания bundled приложения
6. Скомпилирует Inno Setup скрипт для создания установщика

**Результат:** `dist-installer\LegalDesk-Setup-{версия}.exe`

## Установка приложения

1. Запустите `LegalDesk-Setup-{версия}.exe`
2. Следуйте инструкциям мастера установки
3. При первом запуске:
   - Сервер автоматически стартует в фоне
   - Браузер откроется с адресом `http://127.0.0.1:8000`

## Опции установки

При установке можно выбрать:
- **Ярлык на рабочем столе** — создаёт ярлык LegalDesk.exe
- **Автозапуск** — добавляет приложение в автозагрузку Windows

## Управление сервером

### Запуск
- Через ярлык «LegalDesk» в меню Пуск или на рабочем столе
- Автоматически при входе в систему (если выбрана опция)

### Остановка
- Через ярлык «Остановить LegalDesk» в меню Пуск
- Через интерфейс приложения (кнопка выхода)
- При удалении приложения сервер останавливается автоматически

## Удаление

При удалении приложения через «Программы и компоненты»:
1. Сервер автоматически останавливается
2. Все файлы приложения удаляются
3. Запись автозапуска удаляется (если была создана)

## Отладка

Логи работы сохраняются в:
- `%APPDATA%\LegalDesk\logs\launcher.log` — лог лаунчера
- `%APPDATA%\LegalDesk\logs\server.log` — лог сервера

## Troubleshooting

**Python не найден:**
- Установите Python 3.10+ или создайте `.venv` в корне проекта

**Inno Setup не найден:**
- Установите Inno Setup 6 по умолчанию в `C:\Program Files (x86)\Inno Setup 6`

**Сервер не запускается:**
- Проверьте логи в `%APPDATA%\LegalDesk\logs\`
- Убедитесь, что порт 8000 не занят другим приложением
