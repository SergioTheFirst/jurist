# LegalDesk — АРМ Юриста

Веб-приложение для безопасной работы с правовыми базами: автоматическое удаление персональных данных перед отправкой запросов в КонсультантПлюс.

## Возможности

- Анонимизация текста с помощью локальной LLM (Ollama) и regex-правил
- Визуальная проверка замен юристом перед отправкой
- Поиск по правовым базам через API КонсультантПлюс
- Обратная подстановка ПДн в результат

## Быстрый старт

```bash
make install    # установить зависимости
make lint       # проверка кода (ruff + mypy)
make test       # запуск тестов с покрытием
make run        # запуск Flask-сервера
```

## Требования

- Python 3.11+
- Ollama (для локальной LLM)
